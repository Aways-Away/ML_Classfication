# ML_Classfication
手刻线性二元分类器，以两种方法演示：logistic regression 和 generative model。 

二元分类是机器学习中最基础的问题之一，本文中将手刻线性二元分类器，来根据人们的个人资料，判断其收入是否高于50,000美元。

本文将以两种方法：Logistic regression 和 generative model ，来实现二元线性分类。
